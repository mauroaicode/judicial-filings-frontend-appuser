Guía de Alertas Procesales

Entienda el significado de cada color y active el seguimiento automático.

Alerta Crítica

Inactividad +90 días para demandantes o movimientos que requieren acción inmediata.

Alerta Temprana

Inactividad +45 días o trámites procesales de seguimiento obligatorio.

Informativo

Procesos con actividad reciente o inactividad que no afecta negativamente el proceso.

¿Cómo activar el semáforo?

Para que el sistema analice sus procesos, debe vincular su rol (Demandante o Demandado) desde las acciones del proceso o de forma masiva en el módulo de gestión.

Nota importante: El motor procesal analiza automáticamente el contenido y el tiempo transcurrido desde la última actuación para asignar el nivel de alerta correspondiente.
